## Get Started

Clone the repository
and start :
```shell
$ npm i             # Install Node.js components listed in ./package.json
```

### How to use ?

###### iOS:
```shell
$ react-native run-ios
```
Follow up with Xcode

###### Android:
Start your emulator (Genymotion for example)
```shell
$ react-native run-android
```
